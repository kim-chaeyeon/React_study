## 📘 변수와 상수 정리
작성일: 2025.09.26 (금)

### 1. 변수 (let)
```javascript
let age = 24;
```
- 프로그램 실행 중에 값을 바꿀 수 있는 저장소

### 2. 상수(const)
```javascript
const name = "KIM";
```
- 값을 저장한다는 것은 변수와 동일 but **초기화 이후 값 변경이 불가능**

### 3. 변수 명명규칙(Naming Rules)
- $, _ 제외한 기호는 사용할 수 없다.
- 숫자로 시작할 수 없다.
- 예약어는 사용할 수 없다.
